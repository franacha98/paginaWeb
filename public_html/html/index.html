Manual HTML

¿Qué es un Lenguaje de marcas?

- Un lenguaje de marcado o lenguaje de marcas es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen información adicional acerca de la estructura del texto o su presentación.

¿Qué es HTML?

- HTML es un lenguaje de marcas que se utiliza para el desarrollo de páginas de Internet.


ETIQUETA DESCRIPCIÓN
<!DOCTYPE html> Se coloca al principio de todo.
No es una etiqueta de HTML, sino una instrucción que informa al navegador la versión de HTML en la que la página está escrita. No tiene etiqueta de cierre.
<html> Dice al navegador que es un documento HMTL.
Funciona como el contenedor de todos los elementos HMTL, a excepción del <!DOCTYPE html>.
Tiene etiqueta de apertura y de cierre.
Puede contener el atributo lang="es-ES", lo que indica el idioma de la página.
<head> Etiqueta (de apertura y cierre) que contiene todos los elementos head. Estos elementos puedes ser, entre otros:
 - Título
 - Scripts
 - Estilos
 - Metadatos (información sobre la página web)
 - Link
Cabeceras Esta etiqueta representa los diferentes encabezados.
Hay 6 opciones diferentes ([h1 - h6]), que varía el tamaño.
Son útiles para marcar una estructura en nuestra página.
<p> Esta etiqueta define un párrafo.
<br> Etiqueta que no necesita cierre y produce un salto de línea
Formateo de texto Permite dar pequeños estilos al texto.
Los ejemplos más sencillos son:
 - <b>TEXTO</b> ----- TEXTO
 - <i>TEXTO</i> ----- TEXTO
 - <strong>TEXTO</strong> ----- TEXTO
<a> Representa un enlace.
Ejemplo de sintxis: <a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>
Ejemplo de funcionamiento: Visit our HTML tutorial
<img> Esta etiqueta sin cierre muestra una foto del sistema (ruta) o de Internet (URL).
Si la ruta esta mal o la foto no existe, aparecerá en su lugar el atributo 'alt'
Por otro lado, tenemos el atributo 'title', que al poner el cursor encima aparecerá el nombre de la foto.

Listas desordenadas <ul> abre una lista desordenada.
<li> abre cada elemento de la lista. (Con sus cierres respectivos)
  • Coffee
  • Tea
  • Milk
La lista anterior se escribiría así:

<ul style="list-style-type:disc;">
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>
Listas ordenadas <ol> abre una lista desordenada.
<li> abre cada elemento de la lista. (Con sus cierres respectivos)
  1. Coffee
  2. Tea
  3. Milk
La lista anterior se escribiría así:

<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>
Listas de descripción Lista de términos con su descripción.
Coffee
- black hot drink
Milk
- white cold drink
Sintaxis:

<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>
<body> Esta etiqueta define el cuerpo del documento.
Contiene todos los elementos de un documento HTML, como texto, hipervínculos, imágenes, tablas, listas, etc.
<table> Esta etiqueta define una tabla.

El encabezado de la tabla se representa con 'th'.
Cada fila se define con 'tr' (table row).
Cada celda se representa con 'td'.
Ejemplo:

<table>
<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
</tr>
<tr>
<td>Fran</td>
<td>Acha</td>
<td>21</td>
</tr></td>
</table>

Nombre Apellido Edad
Fran Acha 21
<hr> Dibuja una línea horizontal (no tiene etiqueta de cierre).
<button> Crea un botón, pero por sí solo no hace nada, hay que ponerle atributos.
Por ejemplo, para acceder a esta página has hecho click en un botón como este:
Este botón contiene el atributo onclick="window.location.href='index.html'", lo que permite redirigir a otra página.
<pre> Define texto preformateado.
El texto que contiene se muestra en una fuente (normalmente Courier), y mantiene los espacios y los saltos de línea.
(Sin necesidad de <br>)
Estilos Hay tres formas de dar estilos: Inline, internal, external:

Inline:
- Sintaxis: <tagname style="property:value;">
Internal:
- Se añade un tag style en el head. Para ello es necesario que haya identificadores (nombre de un tag (p), id o class)
External:
- Se guardan todos los estilos en una hoja de estilos aparte del .html. Tiene la extensión .css.
Se debe enlazar al fichero html mediante un tag link en el head:
<head>
<link rel="stylesheet" href="hojaEstilos.css">
</head>

Los más comunes son:
- background-color: define el color de fondo del elementos
- color: define el color del texto de un elemento
- font-family:nombre_fuente; define la fuente del texto
- font-size:13px; modifica el tamaño de la letra(lo mejor es poner el tamaño en pixels)
- text-align: modifica el alineado horizontal de un elemento (center, left, right)
Icono Es la imagen pequeña que aparece en la pestaña, al lado del título.
Se coloca en una etiqueta link en el head de la página.
Es recomendable que sea una imagen de tipo PNG de tamaño pequeño, como 16x16 o 32x32.
El ejemplo de esta página sería:
<link rel="icon" type="image/png" href="../images/favicon-32x32.png" sizes="32x32"/>
<address> Define la información de contacto del autor/propietario.
Ejemplo:
<address> Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA </address>
<cite> Define el título de un trabajo.
Los elementos de una cita se muestran normalmente en cursiva.
Ejemplo:
The Scream by Edvard Munch. Painted in 1893.
<cite>The Scream</cite> by Edvard Munch. Painted in 1893.
Bloques Los bloques son la parte fundamental a la hora de maquetar una página web.
Permiten modificar las posiciones y los estilos de las diferentes partes de la página gracias a la agrupación en bloques.
Hay dos tipos de bloques principalmente:
- DIV
Elemento usado como contenedor (bloque) de otros elementos.
Permite dar estilos a todo el bloque que abarca el div.
Ejemplo:

London

London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.

Standing on the River Thames, London has been a major settlement for two millennia, its history going back to its founding by the Romans, who named it Londinium.


- SPAN
Permite agrupar en secciones, pero está más enfocado a textos.
Ejemplo:

My Important Heading
Mapa de imagen Permite poner varios enlaces en una misma imagen, por secciones de la misma.
Es necesario definir un mapa (<map> con areas(<area>) dentro)
Ejemplo:
<img src="images/workplace.jpg" alt="Workplace" usemap="#workmap">
<map name="workmap">
<area shape="rect" coords="34,44,270,350" href="hola.html"/>
</map>
iframes Permite introducir una página web dentro de la misma en un bloque.
Computer code Representa una entrada de teclado para el usuario
Sintaxis: <kbd>TEXTO</kbd>
Guarda el documento pulsando Ctrl + S

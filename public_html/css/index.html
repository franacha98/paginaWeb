Manual CSS En Mantenimiento

¿Qué es CSS?

- Hoja de estilo en cascada. Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado

¿Cómo se aplican los estilos CSS?

INLINE:

- Sintaxis: <tagname style="property:value;">

INTERNAL:

- Se añade un tag style en el head. Para ello es necesario que haya identificadores (nombre de un tag [p], id o class)

EXTERNAL:

- Se guardan todos los estilos en una hoja de estilos aparte del .html. Tiene la extensión .css. Se debe enlazar al fichero html mediante un tag link en el head:
<head>
<link type="text/css" rel="stylesheet" href="hojaEstilos.css"/>
</head>

ATRIBUTO DESCRIPCIÓN
Selectores Sirven para seleccionar un elemento HTML para darle estilos.
  Hay tres formas:

    - Sobre el elemento: ponemos el nombre de la etiqueta HTML y le aplicamos los estilos que queramos a TODOS los elementos iguales.
    Sintaxis: p{text-align: center;
                color: red;}

    - Atributo id: ponemos un id a cualquier elemento html para hacerlo único. Los estilos SOLO se aplican a ese id.
    Sintaxis: #id{background-color: green;}

    - Atributo class: es similar al id, solo que la clase permite agrupar varios elementos a los que queramos darles los mismos estilos.
    Sintaxis: .class:{padding: 10px;}
Padding Genera un espacio de relleno entre el contenido del elemento y lo definido entre los bordes. 
  Se puede definir márgenes a todos los lados del elemento, es decir:
       padding-top: 100px;
       padding-bottom: 100px;
       padding-right: 150px;
       padding-left: 80px;
Margin Genera un espacio fuera de los bordes del elemento para separar los diferentes elementos. 
Se puede definir márgenes a todos los lados del elemento, es decir:
       margin-top: 100px;
       margin-bottom: 100px;
       margin-right: 150px;
       margin-left: 80px;
Colores Se representan de varias formas:
  - RGB(red,green,blue)->RGB(0-255,0-255,0-255)--->Ejemplo: background-color:rgb(255, 99, 71);
  - RGBA(red,green,blue,alpha[transparencia])----->Ejemplo: background-color:rgba(255, 99, 71, 0.5);
  - Por nombre: Click aquí para ver los nombres disponibles
  - HEX (hexadecimal)-> Sintaxis #rrggbb (Números entre 0-9 y letras A-F)--->Ejemplo: background-color: #F0F8FF;

A partir de esto, hay varias propiedades de CSS que permite jugar con estos colores, como por ejemplo:
    -background-color: permite dar color al fondo de un elemento.
    -color: permite dar color al contenido, como por ejemplo el texto, de un elemento.
    -border-color: permite dar color a los bordes de un elemento.
Fondos Permite modificar el fondo de los elementos: 
      - background-color: cambia el color de fondo.
      - background-image: url("image.jpg"); pone una imagen de fondo.
      - background-repeat: permite repetir o no la imagen de fondo.
Bordes Permite modificar los bordes de los elementos: 
      - border-color: cambia el color del borde.
      - border-style: permite cambiar el tipo de borde. Hay varios tipos, como dotted, dashed, solid, hidden, double, etc.
      - border-width: permite modificar el ancho del borde.
      - El borde puede dividirse en 4 partes (top, bottom, right y left), y se le puede dar estilos diferentes a cada una de ellas. Ejemplo:
          #id{
            border-top-style: dotted;
            border-bottom-style: dashed;
            border-right-style: solid;
            border-left-style: double;}
Altura y anchura Permite modificar las dimensiones de los elementos: 
      - width: 100%; cambia la anchura (es recomendable ponerlo en porcentaje).
      - height: 200px; modifica la altura del elemento.
Float Permite flotar elementos para posicionarlos.
